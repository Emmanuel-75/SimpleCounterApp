# Interactive Counter App
A challenge designed to test practical understanding of JavaScript fundamentals by building interactivity into a styled Tailwind UI. 


## Features
- Semantic HTML5 structure
- Clean, readable indentation
- Dynamic DOM manipulation
- Event handling using addEventListener
- Increment and decrement functionality
- Disabled state logic when counter reaches 0
- Addition and subtraction effects
- Reset button to restore default value
- Tailwind-powered styling (centered layout, rounded buttons, shadows)
- Smooth transition effects on value changes

## Tech Stack
- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)

## Purpose
Practical application of JavaScript fundamentals to write simple scripts to make Tailwind-based pages move from static design to interactivity. 

## How to Use
Download the file and run `npm run dev` in the terminal or Tap on the live link associated with this folder
